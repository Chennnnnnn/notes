

### 裴波那契数列

1,1,2,3,5,8,13,21,34

##### 使用循环

```javaScript


function peibo (n) {
    if(n <= 0 ) return 0;
    if(n <= 2 ) return 1;
    var t1 = 1, t2 = 1,sum = 0;
    for(var i = 2; i < n; i ++){
        sum = t1 + t2;
        t1 = t2;
        t2 = sum;
    }
    return sum;
}

```

#### 使用递归

```
function peibo (n) {
    if(n <= 0 ) return 0;
    if(n <= 2 ) return 1;
    return peibo(n-1) + peibo(n-2);
}

```
